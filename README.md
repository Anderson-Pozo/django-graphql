1. Clone the repository
    ```sh
   https://github.com/Anderson-Pozo/django-graphql.git
   ```
2. Create .env file
    ```sh
    DB_HOST= dbpostgres
    POSTGRES_PORT=5432
    POSTGRES_DB=db
    POSTGRES_USER=user_postgres
    POSTGRES_PASSWORD=password
    ``` 
3. In root directory run the command
    ```sh
    docker-compose up -d
    ```
4. Check if containers are running
    ```sh
   docker ps
   ```
